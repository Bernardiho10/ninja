Hub Overview

The Admin Management Hub is the nerve center of the Ninja platform. From here, you can oversee all active business integrations, monitor real-time API traffic, and manage platform-wide configurations.

Note: All actions taken in the Management Hub are logged for security auditing purposes.

Managing Businesses

The Businesses section allows you to view and manage all organizations using the Ninja SDK. Each business record includes:

  • Verification Status: Current KYB level of the organization.
  • API Access: Toggle to enable or disable production keys.
  • Usage Metrics: Aggregate volume of identity lookups.

System Monitoring

Admins have access to the System-Wide Logs, providing a granular view of every request processed by the Ninja engine. This is essential for troubleshooting issues, identifying malicious patterns, and verifying billing data.

Global Settings

Configure core platform behavior, including:

  • Service Fees: Adjust the charge per successful verification.
  • Maintenance Mode: Globally toggle API availability.
  • Support Tiers: Manage helpdesk routing for different business types.
